Google Calendar embed
How to get your calendar embed URL (privacy-safe)
  1. Open Google Calendar → Settings → find your calendar → Integrate calendar
  2. Copy the Embed code (the full <iframe> tag)
  3. In the src URL, add &showTitle=0&showNav=1&showDate=1&showPrint=0&showTabs=0&showCalendars=0&mode=MONTH
  4. Paste just the src URL in the field below — guests will only see blocked/free time
  5. For true privacy, set the calendar's sharing settings to "See only free/busy" in Google Calendar
